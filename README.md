# tcc-clustering
Repositório destinado ao meu trabalho de conclusão de curso (TCC) no curso de Engenharia de Software - UDESC, com o tema: Clusterização de Entidades de Código por Métricas Estruturais: Uma Abordagem Não Supervisionada para Caracterização de Code Smells. Neste repositório contém scripts de dataset de code smells, o arquivo Jupiter Notebook para obtenção das métricas e criação do dataset de métricas e da execução e a análise de aprendizado não supervisionado.
